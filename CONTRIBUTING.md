# Contributing to CoderBlog

First off, thanks for taking the time to contribute! We love to see young developers like you getting involved. 🚀

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an **Issue** and provide details about the problem, including steps to reproduce it.

### Suggesting Enhancements
Got a cool idea? Feel free to open an issue and share your thoughts!

### Pull Requests
Want to write some code? Great! Follow these steps:
1. **Fork** the repository.
2. Create a new **branch** for your feature or fix.
3. Keep your code clean and follow the existing style.
4. Submit a **Pull Request** and wait for a review.

## Code of Conduct
By contributing to this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
